Background

We saw last day that the main repository for bio-molecular structure (the PDB database) only has ~250,000 entries.

UniProtPDB (the main protein seq database) has over 200 million entries.

The EBI AlphaFold database

The EBI AlphaFold database contains lots of computed structure models. It is increasingly likely that the structure you are interested in is already in this database.https://alphafold.ebi.ac.uk/

There are 3 major output from AlphaFold:

  1. A model of structure in PDB format.

2.a pLDDT score: tells us how confident the model is for a given residue in your proteins(high values are good >70)

3.a PAE score: tells us about protein packing quality.

If you canâ€™t find a matching entry for the sequence you are in terested in AFDB you can run AlphaFold yourself.

Running AlphaFold

We will use Colabfold to run AlphaFold on our sequence. https://colab.research.google.com/github/sokrypton/ColabFold/blob/main/AlphaFold2.ipynb
